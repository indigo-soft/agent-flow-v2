# Огляд архітектури

**agent-flow-v2** — це асистент для автоматизації розробки ПЗ на базі ШІ-агентів. Система автоматизує процес від аналізу
ідей до розгортання та документації.

## Основна концепція

Система побудована на взаємодії декількох спеціалізованих агентів, які спілкуються через подійну модель (Event-driven
architecture).

### Основні компоненти:

1. **Dashboard (Frontend)**: Користувацький інтерфейс на Next.js для управління завданнями та перегляду чернеток.
2. **API Gateway**: Центральний вузол для обробки запитів, валідації та маршрутизації.
3. **Агенти (Agents)**:
    * **Architect Agent**: Аналізує бесіди та створює структуровані плани.
    * **Workflow Agent**: Керує життєвим циклом завдань (гілки, мерджі).
    * **Code Review Agent**: Проводить автоматичне рев'ю коду через ШІ.
    * **Documentation Agent**: Підтримує документацію в актуальному стані.
4. **Інтеграції**:
    * **GitHub Integration**: Робота з репозиторіями.
    * **AI Provider**: Взаємодія з LLM (OpenAI, Claude тощо).

## Технологічний стек

* **Backend**: NestJS (Fastify), Prisma (PostgreSQL), BullMQ (Redis).
* **Frontend**: Next.js (React), Tailwind CSS.
* **Monorepo**: pnpm workspaces.

Детальніше про кожен модуль читайте у [modules.md](./modules.md).
